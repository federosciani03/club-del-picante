# 📸 Carpeta de imágenes — Club del Picante

Copiá tus fotos acá con los siguientes nombres:

## Productos
- `miel-chica.jpg`    → Frasco chico 150g
- `miel-mediana.jpg`  → Frasco mediano 300g
- `miel-grande.jpg`   → Frasco grande 500g
- `miel-kit.jpg`      → Kit degustación x3

## Identidad
- `logo.png`          → Logo principal (fondo transparente ideal)
- `logo-white.png`    → Logo versión blanca (para fondo oscuro)

## Galería (opcionales, para la sección "La Experiencia")
- `gallery-1.jpg`     → Foto 1
- `gallery-2.jpg`     → Foto 2
- `gallery-3.jpg`     → Foto 3
- `gallery-4.jpg`     → Foto 4

## Tips
- Formato recomendado: JPG o WebP
- Tamaño máximo recomendado: 500KB por imagen
- Resolución mínima: 800x800px para productos

Una vez que tengas las fotos, actualizá las rutas en:
- `data/products.json` → campo `"image"`
- `public/index.html` → sección galería
